# Singapore Interest Rate and Loan Affordability Dashboard

A banking analytics project using SQL and Power BI to examine Singapore interest rates, commercial lending trends and loan-affordability scenarios.

## Project Status

🚧 In development

The initial repository structure and project plan have been created.

## Project Objectives

- Analyse historical Singapore interest-rate movements
- Examine commercial bank lending trends
- Compare lending activity across industries
- Calculate loan repayments under different interest-rate scenarios
- Communicate financial insights through an interactive Power BI dashboard

## Planned Dashboard Pages

### Market Overview

Historical interest rates, annual changes and headline lending indicators.

### Loan Affordability

Monthly repayment and total-interest comparisons across different loan amounts, interest rates and tenures.

### Industry Lending

Commercial lending trends and comparisons across industries.
